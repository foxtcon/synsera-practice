Architecture decision records

ADR-01
Microsoft-first, in-tenant

All primary controls on Microsoft Security already in E5. Non-Microsoft PAM / IGA only if a hard gap remains after design.

ADR-02
Sentinel lands in the Defender portal

No parallel Azure-portal SIEM path. Migration sequenced in Phase 1 with core detections before MDR option.

ADR-03
Labels before broad Copilot

Proportionate taxonomy and DLP must be live on priority workloads before Copilot expansion beyond pilot.
